## Myflix

**Myflix** mini-Netflix app built using miniflix video library from the **Miniflix** tutorial on **Scotch.io**.

## Installation

```bash

# Get the project
git clone https://github.com/osopanda/myflix

# See Live Demo
myflixdemo2018.surge.sh


# Change directory
cd myflix

# Install the dependencies
yarn install

# Run your app
yarn start
